About the Class

Welcome to our Python coding class, designed specifically for middle schoolers who are curious about the world of programming! This course is perfect for beginners. We'll start with the very basics and guide you through creating your own projects. Learn to code from anywhere with our Zoom sessions!

Class Details

  • Starting Date: June 15th
  • Platform: Zoom (Learn from anywhere!)
  • Target Audience: Middle schoolers (But anyone is welcome!)
  • What you'll need: A computer and an internet connection.

Curriculum

We'll cover the fundamentals of Python programming, including:

  • Strings: Learn how to work with text in Python.
  • Loops: Discover how to repeat actions using for and while loops.
  • Functions: Create reusable blocks of code.
  • Beginning Classes: Get an introduction to object-oriented programming.

The class will culminate in a final project where you can showcase your new skills!

About the Instructors
